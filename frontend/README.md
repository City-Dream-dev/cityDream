# Frontend

#### Dependencies
| Name       | Current Version |
|:-----------|:---------------:|
| npm        |     8.11.0      |
| node       |     16.16.0     |
| next       |     12.2.2      |
| react      |     18.2.0      |
| react-dom  |     18.2.0      |
| typescript |      4.7.4      |


#### Install dependencies
> All commands need to run in the ``frontend`` directory

```json
npm install
```

#### Run dev server

```json
npm run dev
```

#### Build the frontend project

```json
npm run build
```
